Tower Gridlock

One game. Two AIs.
One year apart.

A small experiment to see how far AI coding has come. The same prompt — "build a tower-defense game in a single HTML file" — given to two different AI assistants, twelve months apart. Below: both builds, untouched. Pick one.

Built & deployed at towergridlock.com. Each version is a single self-contained .html file: vanilla HTML, CSS, and JavaScript — no frameworks, no images, no dependencies, no build step.

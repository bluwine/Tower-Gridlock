Tower Gridlock

One experiment. Three AIs.
One file each.

A small experiment to see how far AI coding has come. Gemini and Claude answer the tower-defense prompt directly; Codex takes the single-file constraint as a genre riff and pivots into an Age-of-War lane battle. Below: every build, side by side. Pick one.

Built & deployed at towergridlock.com. Each version is a single self-contained .html file: vanilla HTML, CSS, and JavaScript — no frameworks, no remote assets, no dependencies, no build step. Codex embeds generated art.
